Zadania są pisane w dwóch językach C++ oraz Python.

Plik zad1.cpp należy kompilować w następujący sposób:
a) g++ zad1.cpp -o zad1 
b) ./zad1 ..(i teraz dalsze komendy uruchamiające program w konkretny sposób według polecenia z zadań na liście 2)

Plik draw_chart.py pisany w języku Python może wymagać doinstalowania dwóch bilbiotek: matplotlib oraz numpy za pomocą komend z konsoli:
a) sudo pip install matplotlib
b) sudo pip install numpy

Plik const_nln.py wyznacza na podstawie danych stały wspólczynnik przy nln(n) dla ilości porównań między kluczami dla obu algorytmów: 
- Quick Sort
- Dual Pivot Quick Sort strategia: Count
Dane do tego pliku są umieszczone w pliku data_to_zad3.txt

Plik comp.txt zawiera dane do porównań algorytmów:
- Quick Sort,
- Insertion Sort,
- Merge Sort

Wszystkie wwykresy znajdują się w folderze charts.
Są tam kolejne podfoldery dzielące wszystkie wykresy na konkretne kategorie.
