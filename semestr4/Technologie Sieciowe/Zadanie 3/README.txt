Zadanie należy uruchamiać za pomocą terminala podając odpowiednie argumenty.
Przykład wywołania:

python simulation.py 10 3 3

Pierwszy argument jest długością obiektu Medium.
Drugi argument jest liczbą userów uczestniczących w symulacji.
Trzeci argument jest długością 0-1 ciągów danych wysłanych przez userów.

Program będzie wywoływał co pół sekundy metodę symulującą. 
W oknie terminala będzie pokazywany:
- czas symulacji
- aktualny przebieg sygnałów w obiekcie medium
- id każdego usera
- odebrane dane przez każdego usera
- aktualny status usera
- dane, które user wysyła
- adres usera w obiekcie medium
